Phần 1: Kiến trúc & Công nghệ (Architecture & Tech Stack)

Dựa trên lựa chọn Phương án A (Next.js Serverless & AI Driven), đây là khung sườn kỹ thuật:

Frontend (Giao diện)

Next.js 14+ (App Router): Tốc độ tải trang cực nhanh, hỗ trợ Server Components cho SEO.

Tailwind CSS: Style hiện đại, tối ưu cho di động (Mobile-first).

Backend (Xử lý)

Next.js Route Handlers: Xử lý API tính toán số thiếu ngay trên Edge Runtime (giảm độ trễ).

Gemini API / OpenAI SDK: AI chịu trách nhiệm viết "Storytelling" cho từng cái tên.

Dữ liệu (Storage)

Local JSON / Edge Config: Danh sách tên (Name DB) được lưu trữ dưới dạng tệp tĩnh để truy vấn tức thì.

Ưu điểm

  • Không cần quản lý Server phức tạp.
  • Chi phí vận hành gần như bằng 0 khi mới bắt đầu.
  • Dễ dàng triển khai trên Vercel hoặc Netlify.
